This project investigates whether **nighttime satellite light intensity** can be used as a reliable proxy signal for **economic activity and GDP estimation** across South Africa.

Traditional GDP estimates are authoritative but often come with reporting lags and coarse spatial granularity. Nighttime lights (NTL) observed from satellites have repeatedly been shown to correlate with human activity, electrification, urbanization, and economic output.

The core idea is to learn a mapping from features derived from NTL (and optional auxiliary data) to GDP targets — producing a reproducible, spatially-explicit pipeline that can be monitored over time.
