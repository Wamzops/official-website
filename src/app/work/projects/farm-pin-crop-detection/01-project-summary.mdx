## Introduction

The United Nations' Sustainable Development Goals 2 is “Zero Hunger” by 2030. Hunger and malnutrition remain a significant barrier to development in many regions, particularly in Africa. Approximately 27.4% of the population in Africa was classified as severely food insecure in 2016, which is almost four times as high as any other region.

Accurate data are critical for counties striving to improve food security and agricultural growth.
Sub-Saharan Africa in particular struggles with incomplete or unreliable data on the agricultural sector. In 2018, the international community organized to launch the 50 X 2030 initiative, a commitment to collect better farm data with the goal of ending hunger.

> The goal of this project is to build a robust, reproducible pipeline that classifies crop types from multi‑spectral, multi‑temporal Sentinel‑2 imagery using modern remote‑sensing ML methods, and to produce evaluation artifacts and reusable code that can be applied to agricultural monitoring and decision support.

## Research Questions

1. To what extent can multi-temporal Sentinel-2 spectral bands and derived vegetation indices explain variation in crop types across smallholder fields in South Africa?
2. Which feature engineering choices (spectral bands, vegetation indices, temporal statistics, phenological metrics) are most discriminative for crop type classification?
3. How sensitive are classification estimates to known confounders (cloud contamination, class imbalance, spectral similarity between crops, field boundary misalignment)?
4. What is the best strategy to produce robust crop type predictions when training data is limited and class-imbalanced (temporal cross-validation, data augmentation, transfer learning, ensemble methods)?
5. How do different modeling approaches (traditional ML like XGBoost/Random Forest vs. deep learning like U-Net/LSTM) compare in terms of accuracy, computational efficiency, and generalizability for this task?

#### Expected Key Outputs

<Info color="#00ffff" opacity="0.1">
  - A reproducible GIS and machine learning pipeline for crop type classification using Sentinel-2 imagery.
  - A trained classification model capable of predicting one of nine crop types for each field in the test set.
  - Baseline models (XGBoost, Random Forest) and optionally deep learning models (U-Net, LSTM) for crop type inference.
  - Feature importance analysis to identify the most predictive spectral bands and vegetation indices.
  - A set of maps and visualizations showing classified crop types and model uncertainty.
  - An extensible workflow that can be adapted to other agricultural regions in Africa.
</Info>
