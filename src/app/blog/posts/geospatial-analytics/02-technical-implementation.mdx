# Technical Implementation

This project leverages the computational power of Google BigQuery for geospatial processing. 

### Key Features:
- **Cloud-Native**: Scale analysis across hundreds of gigabytes of satellite imagery.
- **SQL-First**: Utilize BigQuery GIS functions for spatial joins and aggregations.
- **Reproducible**: Fully documented Jupyter notebooks for each step of the pipeline.
